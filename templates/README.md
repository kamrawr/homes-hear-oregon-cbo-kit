# Templates

This folder contains blank templates and examples for CBO operations.

## Available Templates

*Templates will be added as CBOs develop and share their operational documents.*

### Suggested Templates to Add:

- **Intake forms** - Initial household screening questionnaires
- **Eligibility checklists** - AMI verification and program qualification
- **Consent forms** - Data sharing and privacy agreements
- **Workflow diagrams** - Process flows for HOMES vs HEAR pathways
- **Reporting templates** - Data collection and reporting to implementers

## Contributing Templates

When adding templates to this folder:

1. Remove all PII and organization-specific information
2. Use placeholder text like `[Organization Name]` or `[Contact Information]`
3. Include a brief description at the top of the document
4. Name files descriptively (e.g., `intake-form-example.docx`)
5. Save in common formats (.docx, .xlsx, .pdf, or .md)

See `CONTRIBUTING.md` in the root directory for full guidelines.
